PHPUnit runs a TestSuite and returns a TestResult object.

Here is an example:
<?php    require_once 'PHPUnit.php';
class MathTest extends PHPUnit_TestCase {      var $fValue1;      var $fValue2;
function MathTest($name) {        $this->PHPUnit_TestCase($name);      }
function setUp() {        $this->fValue1 = 2;        $this->fValue2 = 3;      }
function testAdd() {        $this->assertTrue($this->fValue1 + $this->fValue2 == 5);      }    }
$suite = new PHPUnit_TestSuite();    $suite->addTest(new MathTest('testAdd'));
$result = PHPUnit::run($suite);    echo $result->toHTML();    ?>
Alternatively, you can pass a class name to the  PHPUnit_TestSuite() constructor and let it automatically add all  methods of that class that start with 'test' to the suite:
$suite  = new PHPUnit_TestSuite('MathTest');    $result = PHPUnit::run($suite);    echo $result->toHTML();
