Class Details



[line 21]

Writes logs file to preserve important information about  framework, can be extended for saving and application actions.

It's created with an array configuration and an indentifier  that defines the log format (what must be saved in logs).  Can handle multiple log resources of different types.  Saves log information on filesystems in CVS format in  different files for each log type.  Saves log requests with the addLogEntry() method
