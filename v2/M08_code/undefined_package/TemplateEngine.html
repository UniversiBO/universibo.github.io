Class Details



[line 15]

Classe astratta che specifica l'interfaccia del template engine.

I comandi tutta via devono essere costruiti rispettando l'interfaccia mostrata  in questa classe, in modo che in futuro possano essere costruite classi Wrapper  per altri TemplateEngines (esempio: xtemplate, pear::template).  Allo stato attuale nel framework viene restituita una istanza di Smarty, che  anche se non eredita questa classe astratta nel rispetta l'interfaccia.
