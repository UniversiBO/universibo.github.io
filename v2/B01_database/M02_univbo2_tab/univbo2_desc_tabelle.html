UniversiBO2: Descrizione tabelle

Di seguito è riportata la descrizione delle tabelle del database Universibo2:

  • canale: contiene le informazioni dei canali (intesi come astrazioni di canali informativi)
  • utente: contiene le informazioni riguardanti gli utenti
  • utente_canale: contiene le informazioni che legano utenti e canali
  • collaboratore: contiene le informazioni riguardo i collaboratori del progetto UniversiBO
  • docente: contiene le informazioni relative ai docenti
  • questionario: contiene le informazioni relative ai questionari compilati dagli utenti per diventare collaboratori
  • file: contiene le informazioni relative ai file
  • file_canale: specifica in quali canali sono stati inseriti i file
  • news: contiene le informazioni relative alle news
  • news_canale: specifica in quali canali sono state inserite le news
  • esami_info: contiene le informazioni degli esami (programma, testi consigliati, appelli...)
  • classi_corso: contiene l'elenco dei corsi di laurea dell'ateneo
  • classi_indirizzi: contiene l'elenco degli indirizzi dell'ateneo
  • classi_orientamenti: contiene l'elenco degli orientamenti dell'ateneo
  • classi_materie: contiene l'elenco delle materie
  • orientamenti: contiene le associazioni orientamenti / corsi / indirizzi e le facoltà a cui appartengono
  • facolta: contiene l'elenco delle facoltà dell'ateneo
  • esami_attivi: contiene informazioni riguardo gli esami attivati
  • sdoppiamenti_attivi: contiene le informazioni riguardo gli sdoppiamenti attivi con riferimento ai mutuati e comuni
